# PokéCup Tournament Site

A simple React + Vite tournament dashboard with three tabs:

- Group Stage
- Playoffs
- Championship Match

## Edit tournament data

Update this file:

```txt
public/tournament.json
```

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

This project is configured for:

```txt
https://nathanrowell.github.io/pokecup/
```

Run:

```bash
npm run deploy
```

Then in GitHub repo settings, set Pages source to:

```txt
gh-pages branch / root
```
